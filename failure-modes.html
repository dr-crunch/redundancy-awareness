Interpretation
Failure modes are not isolated anomalies but repeatable outputs of institutional processes. Each instance links back to its generating relations, completing the causal chain:

Relation → Process → Failure Mode → Legal Condition

This bidirectional structure enables both analysis and evidentiary navigation. Each failure mode entry links to its generating relations above; each relation on the Relations page links back here by anchor.
